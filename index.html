Minhas Habilidades

Python

Python é uma linguagem de programação de alto nível e versátil que pode ser usada em diversas áreas, como desenvolvimento de sistemas, desenvolvimento web, automação de processos e tarefas, análise de dados, inteligência artificial e criação de jogos.

HTML | CSS

HTML é a linguagem de marcação utilizada para criar conteúdo na web, como texto, imagens, vídeos e links. O CSS é a linguagem de estilo utilizada para definir a aparência visual do conteúdo, permitindo a definição de cores, fontes, tamanhos e outros aspectos visuais.

Django

Django é um framework de desenvolvimento web em Python que permite a criação de aplicativos web de maneira rápida, eficiente e segura. Com ele, é possível gerenciar a estrutura do site, o banco de dados, os formulários, as páginas, entre outras funcionalidades.

VBA

O VBA é uma linguagem de programação usada principalmente para automatizar tarefas e processos em aplicativos como o Excel e Word. Com ele, também é possível controlar o SAP GUI Scripting, permitindo criar formulários personalizados e automatizar tarefas e relatórios.
